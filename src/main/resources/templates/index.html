JobTrigger is a Edison Microservice. It's purpose is to discover other Microservices that provide job definitions (just like edison microservices do), and to trigger these jobs according to this definition.

Jobs are used to regularly execute background tasks like importing data from other systems. Using a JobTrigger, only one instance of a possibly load-balanced cluster of microservices will run the job.
